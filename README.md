# TIC TAC TOE (Python - Jupyter Notebook)

A simple **Tic Tac Toe game** implemented in **Python**, designed to run in a **Jupyter Notebook**.  
This project allows two players to play Tic Tac Toe in a console-based interface.

---

## Features
- Two-player mode (Player X vs Player O)
- Dynamic board updates after each move
- Input validation to prevent invalid or duplicate moves
- Automatic detection of win, loss, or draw outcomes
- Runs directly in Jupyter Notebook without additional setup
